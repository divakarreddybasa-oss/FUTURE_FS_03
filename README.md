# Invas Multicuisine Restaurant - Premium Website

A high-end, visual-first digital experience designed for **Invas Multicuisine Restaurant**. This website combines modern aesthetics with robust functionality to deliver a premium brand presence.

## ✨ Key Features

- **🏆 Premium Design**: Sophisticated dark-mode aesthetic with gold accents and glassmorphism UI elements.
- **🖼️ Visual Menu**: Interactive menu tabs (Starters, Main Course, Desserts) with high-resolution food photography and stylized dietary tags.
- **🛰️ Satellite Mapping**: Integrated full-color, high-detail satellite maps with direct "Open in Google Maps" CID linking.
- **📱 Social Connectivity**: Suite of branded interactive social buttons and direct WhatsApp integration.
- **✨ Smooth Animations**: Powered by **AOS (Animate On Scroll)** for a seamless, professional feel.
- **🎨 Custom Branding**: Typographic logo design and a custom restaurant-themed favicon.
- **🖱️ Slim Scrollbar**: A custom slim, gold-themed scrollbar for enhanced desktop experience.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Design System)
- **Logic**: Vanilla JavaScript (ES6+)
- **Animations**: [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- **Icons**: [FontAwesome](https://fontawesome.com/)
- **Typography**: Playfair Display (Serif), Plus Jakarta Sans (Sans-Serif)
